## 🚀 Getting Started

### 🔧 Run Locally

1️⃣ **Install dependencies**  
npm install

2️⃣ **Start the development server**  
npm run dev

#### The app will be available at: [http://localhost:3000](http://localhost:3000)

3️⃣ **Build for production**  
npm run build

4️⃣ **Run tests locally**

**Unit tests**

npm run test
